FlexBans

FlexBans is a multi-platform plugin for Minecraft servers that allows managing player punishments via a web interface.

Buy on BuiltByBitDocumentation

Context & Subject

As part of the development of a Minecraft server, the idea of changing the in-game moderation method, which was managed by a paid and unsuitable plugin, emerged. In 2023, I therefore started the development of FlexBans, which, with a huge practical difference, possesses a web interface allowing any moderation action to be performed without logging into the game. To date, development is not finished but a majority of key features are already present.

Strategic Objectives

01

Centralize player punishments on a network of servers or on a single server.

02

Provide a web interface for moderators to view history and punish players.

03

Allow the use of DBMS such as H2 and SQLITE (file-based) and MySQL (requiring MySQL server).

04

Offer an API accessible by any other plugin allowing interaction with FlexBans.

Key Stages

Requirements Analysis

Surveys for my server staff members to understand the weak points of the already present plugin.

DB and Web Interface Thought Process

Creation of the database schema as well as the design of the web interface.

Plugin Development

Writing the main logic in Java and implementation of Paper API, BungeeCord, and Velocity events.

Web Interface

Development of the React interface and implementation of the REST API for the backend and logic.
